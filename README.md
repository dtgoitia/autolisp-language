## Quick start

Scan dates:

```shell
gpy scan date path/to/file/or/dir
```

<!--
Scan GPS coordinates:

```shell
gpy scan gps path/to/file/or/dir
```
-->

Edit file metadata

```shell
gpy meta file path/to/file/or/dir
```

## Install

Repository developed using Python 3.9.

```shell
make install
```

Assumption: `exiftool` is already installed.
